1. Compiler not found.

  If you get the error message:

    "Check path for nmake and mspdb60.dll"

  you maybe have not set the environment variables required to find the
  compiler.  You can usually set them starting the batch file

    VCVARS32.BAT

  found in the VC98/Bin directory of the Microsoft Visual Studio.
